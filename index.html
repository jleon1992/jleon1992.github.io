Story

After being introduced to web development by a good friend, I quickly became fascinated and dedicated to full-stack web development. I am quickly gaining experience, expanding my skillset, and getting daily practice with project based learning, and am having plenty of fun along the way! I am truly grateful to have found a career which I love. Below are projects listed in chronological order that show both my skill as a web developer but the progress I have made in a short 3 months.

Landing page for a fictional application called WunderList2.0. The purpose of this project was to use all the HTML, CSS and JavaScript we learned in the unit to create a clean landing page.

Food Truck Tracker
